# LAZIZA va LACHO Runner

Bu loyiha siz so‘ragan mini runner o‘yinining oddiy web-versiyasi:

- `LAZIZA` — ro‘moldagi qizaloq
- `LACHO` — oddiy qizaloq
- Qahramon oldinga harakat qiladi, o‘yinchi sakrash orqali to‘siqlardan o‘tadi
- Ball yig‘ib, ketma-ket vazifalar bajariladi

## Ishga tushirish

1. Loyihani oching.
2. `index.html` faylini brauzerda oching.

Yoki terminaldan:

```bash
python3 -m http.server 8000
```

So‘ng brauzerda `http://localhost:8000` ni oching.

## Boshqaruv

- `Space` yoki `↑` — sakrash
